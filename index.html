Introducing, Brainstem.

Brainstem is a fast, scalable, and efficient set of REST-ful web services used to help make Amazon's Web Services easier to manage. These web services serve as a “broker" between end users and developers as they create, modify, and remove various pieces of AWS infrastructure, ensuring that critical business needs (security, authorization, configuration, cost control, etc) are fulfilled. Users, developers, and engineers are able to choose between making Brainstem API calls directly, or using a web GUI built on top of the API.

Brainstem is a REST API written in Ruby. Brainstem contains detailed built-in documentation that explains the usage for each of the available API calls, including sample code for many popular programming languages, to get you up and running as soon as possible. We use AWS everyday and we know what it takes to get things done. Brainstem helps us get those things done faster and more efficiently.

Brainstem Features

Active Directory Integration

Brainstem integrates directly with Active Directory, allowing authentication and authorization to be centrally managed in one place. Why have another system to manage your system?

Multiple AWS Services

Brainstem currently supports EC2, EBS, EMR, RDS, and IAM. These are the services that get the highest usage, so we included them first. We are continually developing new features and adding new services to Brainstem.

Temporary Resource Access

Brainstem provides support for granting access to specific AWS resources for a temporary amount of time. This allows admins to give temporary access to a third party without having to worry about revoking the privileges down the road. The temporary access can also be tailored to exactly the resources needed, and nothing more, by using a pre-defined set of access policies. This also allows developers to programatically request a set of keys instead of hardcoding them or storing them in their script or application. By requesting these temporary Amazon API Keys, developers are still able to utilize all of Amazon's Web Services.

Logging

Every request made through Brainstem is logged. This allows admins the ability easily audit changes made to their cloud infrastructure. Retrieving the logs is as easy as making a REST API call.

REST Easy

One of the great benefits of using a REST interface is that it can be called from any language that can make HTTP requests. To make it even easier, we have provided sample code for some of the most common languages in use here at VivaKi, including Ruby, C#, Java, Python, Javascript, PHP and Curl.

Smart Caching

Smart caching of AWS data provides fast API response times for users of brainstem. Typically, requesting information about AWS resources via brainstem is faster then querying that same information directly from Amazon.

Brainstem Web Interface

The Brainstem Web Interface provides a clean and simple way to provide control your AWS resources. The web interface is written in HTML5, Javascript, and CSS. This allows the interface to be extremely fast and flexible. Since the Web Interface is written entirely in client side code it actually makes the same API requests that are available to any user or developer.

One of the special features in the Brainstem Web Interface is the “Develop Mode” option. This option allows a user to see the raw API request before it's sent to the Brainstem API server. This will hopefully lower the learning curve for developers wanting to use Brainstem programmatically.
